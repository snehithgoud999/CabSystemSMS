# CabSystemSMS

CabSystemSMS is a Spring Boot backend application used to manage cab booking and user information.

## Technologies Used
- Java
- Spring Boot
- Maven
- MySQL
- REST API

## Features
- User registration
- Cab booking system
- Database storage
- RESTful APIs

## How to Run
1. Clone the repository
2. Run `mvn clean install`
3. Run the application using:
   java -jar target/cabsystemsms.jar
